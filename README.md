# chloe‑code – Assistant IA local pour le développement

## Description
Assistant IA open‑source qui génère, refactorise, teste et documente du code depuis VS Code. Fonctionne entièrement hors‑ligne, utilise Ollama (Llama‑2‑13B‑Chat 4‑bit) et des sandboxes Docker.

## Installation rapide
```bash
./install.sh
```
## Usage
Lancez VS Code, ouvrez la palette de commandes (⇧⌘P) et cherchez “Chloe‑code”.
